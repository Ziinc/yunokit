# API Reference

## Authentication

### Get API Token
```typescript
POST /api/auth/token
{
  "email": string,
  "password": string
}
```

### API Key Management
- Generate API keys
- Revoke keys
- Set permissions
- Track usage

## Content API

### List Content
```typescript
GET /api/content/:type
Query Parameters:
- page: number
- limit: number
- sort: string
- filter: object
```

### Get Single Item
```typescript
GET /api/content/:type/:id
```

### Create Content
```typescript
POST /api/content/:type
{
  "fields": object,
  "status": string,
  "locale": string
}
```

### Update Content
```typescript
PATCH /api/content/:type/:id
{
  "fields": object
}
```

## Media API

### Upload File
```typescript
POST /api/media/upload
Content-Type: multipart/form-data
```

### Get Media Info
```typescript
GET /api/media/:id
```

## GraphQL API

### Schema
```graphql
type Query {
  content(type: String!, id: ID): Content
  contents(type: String!, filter: FilterInput): [Content]
  media(id: ID!): Media
}

type Mutation {
  createContent(type: String!, input: ContentInput!): Content
  updateContent(type: String!, id: ID!, input: ContentInput!): Content
  deleteContent(type: String!, id: ID!): Boolean
}
```

### Subscriptions
```graphql
type Subscription {
  contentUpdated(type: String!): Content
  mediaUploaded: Media
}
```

## Rate Limits

### Default Limits
- 1000 requests/hour for REST API
- 100 requests/minute for GraphQL
- 50MB/hour for media uploads

### Headers
```
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Reset: 1623456789
```

## Error Handling

### Error Format
```typescript
{
  "error": {
    "code": string,
    "message": string,
    "details": object
  }
}
```

### Common Status Codes
- 400: Bad Request
- 401: Unauthorized
- 403: Forbidden
- 404: Not Found
- 429: Too Many Requests
