# YunoFeedback

Collect product feedback and vote on features. Issues track votes and comments using YunoContent components.

## Integration

Add `yunofeedback` to your project's `config.toml` under `api.schemas`.

Relevant tables:

- `yunofeedback.issues`
- `yunofeedback.votes`
- `yunofeedback.tickets`
- `yunofeedback.ticket_comments`
