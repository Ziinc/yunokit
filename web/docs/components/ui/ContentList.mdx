## Usage

Use the `data` key to pass data into the component.

The `itemTitleKey` and `itemDescriptionKey` are used to determine what values to display for each item.

## Props

| Name                 | Type                  | Required? | Description                                                                              |
| -------------------- | --------------------- | --------- | ---------------------------------------------------------------------------------------- |
| `data`               | `ContentItem[]`       | Required  | `ContentItem`s to render                                                                 |
| `itemTitleKey`       | `string`              | Required  | The key of the content item's `data` field to use to render the title of the item.       |
| `itemDescriptionKey` | `string`              | Optional  | The key of the content item's `data` field to use to render the description of the item. |
| `itemClassName`      | `string`              | Optional  | The `className` of the content item.                                                     |
| `className`          | `string`              | Optional  | The `className` of the content list.                                                     |
| `style`              | `React.CSSProperties` | Optional  | The `style` of the content list.                                                         |
