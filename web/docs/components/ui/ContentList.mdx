# `<ContentList />`

Displays a list of content items.

<ComponentDemo Component={ContentList} props={PROPS} />

## Usage

The `itemTitleKey` and `itemDescriptionKey` are used to determine what values to display for each item.

## Custom Styling

ContentList ships with basic styling out-of-the-box. These styles can be overridden by a variety of methods.

You may provide custom classes to each rendered element, such as by using

<ComponentDemo Component={ContentList} props={PROPS} />
